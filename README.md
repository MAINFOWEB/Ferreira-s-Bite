# Ferreira's Bite - Landing Page

## 🚀 O Projeto
O Ferreira's Bite é uma aplicação web de alta fidelidade que funde a estética nostálgica dos Diners americanos com o futurismo do Cyberpunk. Desenvolvido com foco em UX imersiva e performance visual.

## 🛠 Tecnologias
- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização avançada, efeitos de Glow/Neon (box-shadow) e design responsivo.
- **JavaScript (Vanilla):** Manipulação dinâmica de elementos e interatividade.

## 🎨 Principais Características
- **Interface Dark Mode:** Contraste elevado fiel ao conceito Cyberpunk.
- **Responsividade:** Layout adaptável (Mobile-First).
- **Otimização:** Assets e animações otimizados para carregamento rápido.

## 💻 Como Rodar
1. Clone o repositório:
   `git clone https://github.com/seu-usuario/ferreiras-bite.git`
2. Abra o arquivo `index.html` diretamente no seu navegador ou utilize a extensão "Live Server" no VS Code.
