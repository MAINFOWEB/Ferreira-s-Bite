# Projeto de Edificação - Planejador Residencial

## 🚀 O Projeto
O Projeto de Edificação é uma aplicação web de alta precisão voltada para o planejamento arquitetônico e simulação de espaços. Fundindo uma interface técnica de CAD com a usabilidade de uma aplicação moderna, o sistema permite desenhar plantas, gerenciar estoques de componentes e visualizar terrenos em 2D ou vista lateral, tudo de forma intuitiva e performática.

## 🛠 Tecnologias
- **HTML5:** Estrutura robusta para manipulação de canvas e interfaces.
- **CSS3:** Estilização responsiva e design técnico funcional.
- **JavaScript (Vanilla):** Lógica matemática de renderização, manipulação de estados, cálculos de área e automação de desenho.

## 🎨 Principais Características
- **Canvas Dinâmico:** Renderização precisa de cômodos, escalas e elementos técnicos.
- **Gestão de Inventário:** Estoque inteligente para controle de ativos e componentes.
- **Visualização Dual:** Alternância fluida entre planta baixa e vista lateral técnica.
- **Selo de Projeto:** Geração automática de selos técnicos com dados de localização e quadro de áreas.
- **Controles de Precisão:** Atalhos para rotação, redimensionamento e posicionamento de elementos.

## 💻 Como Rodar
1. Clone o repositório:
   `git clone https://github.com/seu-usuario/projeto-edificacao.git`
2. Abra o arquivo `index.html` diretamente no seu navegador ou utilize a extensão "Live Server" no VS Code para uma experiência de desenvolvimento em tempo real.
